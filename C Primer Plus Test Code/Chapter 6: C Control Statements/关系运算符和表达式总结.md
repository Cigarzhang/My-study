### 关系运算符：
每个关系运算符都把它左边的值与它右边的值进行比较。
| 运算符   |   含义  | 
|---------|---------|
|  <  | 小于        | 
| <=  | 小于或等于   |
| ==  | 等于        |
| >=  | 大于或等于   |
| >   | 大于        |
| !=  | 不等于      |
### 关系表达式：
  一个简单的关系表达式由一个关系运算符及其两侧的操作数组成。如果关系为真，关系表达式的值为1；如果关系为假，关系表达式的值为0/
### 例如：
- 5 > 2 为真，则该关系表达式的值为1.

- (2 + a) == a为假，则该关系表达式的值为0.
